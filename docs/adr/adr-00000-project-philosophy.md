# Project Philosophy
## Status
Accepted

Date: 2026-07-14

Decider(s): Aaron Mina

## Decision
The RPG Engine will prioritize maintainability, readability, extensibility, and correctness over development speed.

## Consequences
1. Good - Encourages thoughtful design
2. Good - Makes onboarding contributors easier
3. Good - Results in a more maintainable codebase
4. Bad - Slower initial development
5. Bad - More documentation and design work
